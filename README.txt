Step 1: install python 3.7
Download it from https://www.python.org/downloads/ or run the file python-3.7.0.exe (under the Installers folder)

Step 2: run the file install_pyperclip.bat (under the Installers folder)
# This just runs "python -m pip install pyperclip" 
On a Mac you may need to run: "python3 -m pip install pyperclip"

Step 3: go to the yadvashem page you would like to copy and press control + a

Step 4: run convert_control_clipboard.py 

Step 5: paste your newly converted data as a source!
